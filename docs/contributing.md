# Contributing

Thank you for helping improve BenchmarkingBrews! Contributions of all sizes are welcome.

## Development setup
1. Install the [PlatformIO IDE](https://platformio.org/) extension for VS Code or the CLI.
2. Run `pio run` from `firmware/` to build the firmware.
3. Open `web-tool/index.html` in a Web Bluetoothâ€“capable browser for quick UI testing.

## Coding standards
- Keep files formatted with the PlatformIO/Arduino defaults.
- Favor small, testable classes and explicit pin definitions in `config.h`.
- Include documentation updates in `docs/` when adding new hardware features.
